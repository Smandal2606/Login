Customer Backend Login System (Spring Boot)

This project is a Customer Backend Login System developed using Spring Boot. It provides authentication functionality where existing customers can log in using their credentials, and new users can register themselves.

Features
Customer registration for new users
Secure login using username and password
Backend REST APIs built with Spring Boot
Database integration for storing customer details
Validation of user credentials
Technologies Used
Java
Spring Boot
Spring Data JPA
Hibernate
MySQL Database
REST API
Functionalities
Register Customer
New users can create an account by providing:
ID
Username
Password
Customer Login
Existing customers can log in using their username and password.
System validates credentials from the database.
